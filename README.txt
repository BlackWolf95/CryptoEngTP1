To execute the given code, extract the tar ball and please do the following on the terminal:
1. make : to compile the C source codes and create the executables.
2. ./AES2 : to execute the code to generate the previous round key from the last round key (according to the FIPS document)
3. ./AES3 : to execute the F function along with the distinguisher
4. ./AES4 : to execute the key recovery attack on 3 1/2 rounds of AES128 bit encryption.
5. make clean :  to delete all the executables and target files.
